#EXE2X GUI

Executable (.EXE) to script files converter:
- Visual Basic (.VBS and .VBE)
- JScript (.JS and .JSE)
- Batch (.BAT and .CMD)
- Windows Script (.WSF)

with graphical interface for simplicity, small and simple.
